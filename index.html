Earthquake Map

This map is a simple data visualization project that displays real-time earthquake data from an Atom feed at the U.S. Geological Society. The map(s) come from Unfolding Maps, and the legend and graphics are made with Processing 2.0. This project was part of the coursework of a Coursera Java class called Data Structures and Performance offered by UCSD.

Shortest Path Algorithm

This was another project for a Coursera class from UCSD called Advanced Data Structures in Java. Google Maps were used to extract data points that were parsed into intersection points, and shortest path algorithms were used to find the shortest route between two points. Algorithms for this project included BFS, Dijkstra, and A*.

I also created a modified version that took speed limits into account and generated the fastest route between two points. Since speed limit data is unavailable for free, I identified road segments as per Open Street Map and assigned approximate speeds per segment.

ESL Mobile Movie Site

One of my careers is ESL, and I currently teach at Santa Monica College. I started a movie club for the ESL students, and after I realized my study guides reached upwards of 30 printed pages per movie per person, I decided to save a forest and put the guides in a mobile web site.

Since all of my students are surgically attached to their cellphones anyway, and since this was a great way to practice my skills, the web site seemed like a great practical solution. My students and my colleagues both gave it positive reviews. Clicking on the image will redirect you to the actual site, which is best viewed on a mobile device. Each movie image links to that movie's study guide.

The site is mostly static HTML with some minor CSS and Bootstrap navigation. It is clean, user-friendly, and easy to read.

Wikipedia Page Viewer

A Wikipedia Page Viewing project as part of the projects for freeCodeCamp. I used AngularJS for the calls to the Wikipedia API. The user can either click the link to get a random page or use the search engine to get the top ten results of a custom search. Results are displayed with thumbnail images (if they exist on the Wikipedia page) in a sequence of custom outputData objects.

Simon Game

A Simon Game project as part of the projects for freeCodeCamp. The play sequence is for twenty turns. If strict mode is on, the game resets once a mistaken button is pushed. The error face and happy face will likely appear out of place on various browser settings. The error handling code mostly works, but if you get click-happy, the timing will break.
